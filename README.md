# Calendar-example

## Setup

1. Clone the repository:
   ```
   git clone <repository-url>
   cd Calendar-example
   ```
2. Install dependencies:
   ```
   npm install
   ```
3. Start the development server:
   ```
   npm run dev
   ```
4. Build the project:
   ```
   npm run build
   ```
5. Preview the production build:
   ```
   npm run preview
   ```
